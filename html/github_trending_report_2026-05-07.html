AAAI 2026 录用论文开源实现，第一个在 45 个全球交易所 K 线数据上预训练的开源金融基础模型。两阶段框架：① 专属 Tokenizer（将 OHLCV 连续数据量化为层次化离散 token）② 大型自回归 Transformer（在金融 K 线"语言"上预训练）。模型家族：Kronos-mini / Kronos-small / Kronos-base（Hugging Face 开放），内置 A 股微调脚本，Live Demo 展示 BTC/USDT 未来 24 小时预测。

✨ 核心特性

  • 专属 Tokenizer：OHLCV → 层次化离散 token，专为金融时序设计
  • 预训练数据：45+ 全球交易所，股票/加密货币/期货覆盖
  • 模型家族：Kronos-mini / Kronos-small / Kronos-base（HuggingFace）
  • 支持任务：多空预测 / 价格预测 / 趋势识别 / 波动率预测
  • A 股微调：内置 4 步微调脚本（Configure→数据→微调→回测）
  • AAAI 2026 学术背书，arXiv 论文 2508.02739

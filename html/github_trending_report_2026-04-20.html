WiFi 感知突破：RuView 让无线信号看穿墙壁

ruvnet/RuView 以 47,505 Stars 位居今日星数之首。它用房间里现有的 WiFi 信号（CSI 信道状态信息）实现人体姿态估计、呼吸率/心率监测和位置检测，整套系统完全不需要摄像头。因为没有视频像素，GDPR 视频合规和 HIPAA 成像合规都可以绕过，医疗、零售、楼宇管理场景天然合规。$8 的 ESP32 模块 + Docker 一条命令即可部署，17 种预训练感知应用无需重新训练，v0.7.0 已可用。

Claude 生态正在从"单个工具"变成"AI 工作室"

FinceptTerminal：开源版 Bloomberg Terminal 今日登顶

"数据自主"叙事：Paperless-ngx 和 thunderbolt 双双上榜

Paperless-ngx（38,877 Stars）今日重新进入 Trending，用 OCR + 自动分类把文档变成可搜索的在线档案馆，docker compose 一条命令部署，147 个版本迭代。thunderbolt（2,233 Stars，24 小时涨约 600）持续增长，定位"你控制的 AI 客户端"。两个项目都属于"自托管 + 数据不上云"赛道，在对大平台数据隐私日益警惕的当下持续获得社区关注。

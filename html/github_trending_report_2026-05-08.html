🏛️ github/spec-kit（93k）——GitHub 官方以平台级力度背书"Spec-Driven Development"，直接狙击 vibe coding

GitHub 官方出品 spec-kit（93,239 ⭐ + 8,089 forks + 142 releases），今日毫无悬念占据日榜第一。核心观点：Spec-Driven Development 让规格书变成可执行的，直接生成实现，替代"vibe coding"。6 步工作流对应 /speckit.* slash command，specify CLI 支持 30+ AI coding Agent（Claude Code / Cursor / Windsurf 等），社区 Extensions / Presets 生态已成型。这完美接续本周 karpathy-skills（哲学原则）→ agent-skills（工程 Skill）→ spec-kit（规格可执行）的三层叙事——AI coding 范式从"感觉驱动"向"规格驱动"的平台级迁移正式开始。

🦆 aaif-goose/goose（44.6k）——Block（Jack Dorsey）的开源 AI Agent 入驻 Linux Foundation，Agent 赛道里程碑

goose 是 Block 公司旗下、现已转入 Linux Foundation Agentic AI Foundation（AAIF）的开源通用 AI agent，44,643 ⭐ + 4,564 forks。不只是代码助手：研究/写作/自动化/数据分析全覆盖；桌面+CLI+API 三端；Rust 构建；15+ LLM Provider；70+ MCP 扩展。goose 进入 Linux Foundation 是今日最重要的生态信号：AI agent 从公司私有生态走向开放基金会治理，类比早年 Kubernetes 进入 CNCF，Agent 赛道正式进入"基金会时代"。

🗂️ VectifyAI/PageIndex（29.9k）——"vectorless RAG"挑战传统向量检索，FinanceBench SOTA 98.7%

核心挑战：similarity ≠ relevance，向量相似度无法代替真正的相关性。PageIndex 受 AlphaGo 启发：① 为文档生成"目录树"层次索引 ② LLM 通过树搜索推理式检索，无向量库 + 无分块，FinanceBench 达到 98.7%（SOTA）。今日与 github/spec-kit 同框，今日 Trending 的主旋律是 "reasoning over vibe"：spec-kit 用规格推理替代感觉编程，PageIndex 用文档推理替代向量相似度检索。

📊 今日大换血：11/12 新上榜，"机构开源日"——GitHub / Block-AAIF / AWS / Anthropic / OpenAI / Vercel 同日集中释放

今日 Trending 最显著特征：11/12 新上榜（近期最高换血率），且多来自顶级机构。GitHub 推出 spec-kit，Block/AAIF 的 goose，AWS 的 aidlc-workflows（Explore），Anthropic 的 financial-services，OpenAI 的 plugins（Explore），Vercel 的 open-agents（Explore）同日上榜。当大厂集中释放 AI 工具栈时，往往意味着某个方法论（今日：spec-driven + agent governance）已从"社区自发探索"进入"平台级标准化"阶段。

  • Vision RAG：直接在页面图像上推理，无需 OCR

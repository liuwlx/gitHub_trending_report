🏆 "超级资源三件套"同日登场——public-apis（432k）+ coding-interview-university（346k）+ markitdown（121k）

三大知识资源类项目同日 Trending，合计星数超 90 万。共同特征是"无业务逻辑、纯粹输出价值"：public-apis 是 500+ 免费 API 集合、coding-interview-university 是 CS 系统学习路线图、markitdown 是微软文档转 Markdown 工具。三者同日登榜是社区学习与技术分享热潮的最强信号：当大量开发者同时分享"让你变强"的资源时，Trending 就会出现这种集体共振。其中 coding-interview-university 的 82k forks 是今日所有项目最高，public-apis 432k Stars 是 GitHub 全站前 5。

🤖 forrestchang/andrej-karpathy-skills（115k）——"一个 CLAUDE.md 文件"引发的 GitHub 现象级事件

115,419 ⭐、11,544 forks，仓库核心内容是一个 CLAUDE.md 文件。作者将 Andrej Karpathy 批评 LLM 编程缺陷的帖子提炼为 4 条原则：① Think Before Coding（主动问清需求，不默默假设）② Simplicity First（最少代码，拒绝过度设计）③ Surgical Changes（只改必须改的，不顺手优化）④ Goal-Driven Execution（每步定义可验证的成功标准）。不是工具，不是框架，是一个行为约束文件——却在 Claude Code 用户社区病毒传播到 115k Stars。这是"Karpathy 品牌 × AI 编程工具爆发"最具代表性的交叉现象。

🦌 bytedance/deer-flow 2.0（65k）——字节跳动 SuperAgent Harness 重磅开源

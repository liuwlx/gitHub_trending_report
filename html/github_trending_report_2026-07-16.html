💡 今日洞察

Agent 技能包继续占据头部

skills、hallmark、marketingskills 都在把经验整理成可安装、可组合的工作流。竞争点正从“模型会不会写”，转向“流程能不能稳定复用”。

低成本模型开始拼工程框架

Open Interpreter 转向 Rust、Codex 派生架构和多 Harness 切换。它不是换聊天皮肤，而是在研究便宜模型怎样借助更强执行框架把活干完。

自托管 AI 产品长成完整系统

AIRI 与 DeepTutor 都包含前端、运行时、工具、状态、部署和多端入口。这类仓库已经不是“一份 Prompt 配三张截图”。

第一名也得看成熟度

OpenCut 热度最高，但当前仓库仍是重写中的新架构。热榜第一说明大家都在看，不等于今晚就该替换生产工具。

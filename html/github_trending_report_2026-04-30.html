Fully automatic censorship removal for language models. 基于方向性消融 + Optuna TPE 优化器，全自动寻找最优消融参数，社区已产出 1000+ Heretic 模型。
⭐20,245 🍴2,072 Python 📄 MIT
llm abliteration censorship-removal optuna transformer

📖 项目摘要

Heretic 是一个全自动 LLM 安全对齐消除工具，基于 Arditi et al. 2024 方向性消融理论，结合 Optuna TPE 参数优化器，在共同最小化拒绝率和与原始模型 KL 散度的条件下自动找到最优消融参数。无需理解 Transformer 内部结构即可使用，支持大多数稠密模型、多模态模型和部分 MoE 架构，社区已基于 Heretic 发布超过 1000 个消融模型。

✨ 核心特性

  • 全自动：Optuna TPE 优化器自动搜索参数，无需人工干预
  • 双目标优化：最小化拒绝率 + 最小化 KL 散度（保留模型能力）
  • 参数化方向消融：direction_index / max_weight / min_weight 等可学习参数
  • 支持大多数稠密模型、多模态模型，部分 MoE 架构
  • HuggingFace 社区：已产出 1000+ Heretic 消融模型（Gemma-3 / GPT-OSS 20B / Qwen3 等）

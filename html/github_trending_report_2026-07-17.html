🔍 今日洞察
🧰 AI 编程工具开始比拼“运行底盘”

Open Interpreter、LobeHub、DeepTutor 和两套工程 Skills 都不再满足于给模型套一层聊天界面。它们在争夺的是会话状态、工具调用、权限、长期任务、知识库和多 Agent 编排。模型仍重要，但决定用户体验的越来越是 Agent harness 和运行时工程。

🪴 本地小模型从“能跑”走向“能做事”

Bonsai Demo 把 1-bit / ternary 模型、llama.cpp、MLX、视觉输入、OpenAI 风格工具调用和 MCP 放进一套安装脚本里。开发者关心的不只是压缩率，而是普通设备能否真正跑起长上下文、视觉和工具链。

🎬 开源生产力工具一边爆发，一边重写地基

OpenCut 的当日增量很高，但当前主仓库明确处于从零重写阶段，经典版本才是现阶段更稳妥的使用入口。星标像掌声，地基像钢筋，鼓掌再响也不能替钢筋上班。

🧭 数据、规范与技能库也能成为基础设施

Apache Ossie 试图统一分析、AI 与 BI 的语义元数据；exercises-dataset 把健身数据、Schema 和浏览器打包；Hallmark 与 mattpocock/skills 把设计和工程方法做成可分发的 Agent Skill。

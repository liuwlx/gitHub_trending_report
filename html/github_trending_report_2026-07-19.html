🗺️ 流式三维重建登顶，重点从“拍完再算”转向“边走边建图”

Robbyant/lingbot-map 以 831 Stars Today 排名第一。它用流式 Transformer、关键帧和 KV Cache 持续更新场景表示。对机器人、长视频和大尺度空间建模很有吸引力，不过约 20 FPS 与 SOTA 都是官方结果，先别拿宣传海报当显卡验收单。

📚 今日增星冠军还是“自己造一个 X”，基础功又回来抢镜

build-your-own-x 一天新增 1,126 Stars，是榜单增速最高项目。大家一边用 Agent 写代码，一边又开始补数据库、编译器、容器和浏览器原理。它是教程索引，不是软件系统，不能给目录硬画微服务图。

🔌 Agent 工具开始围着协议、上下文和本地数据打磨

code-review-graph 用结构图减少无效上下文，wigolo 把搜索、抓取、爬取和研究封装成 MCP，kimi-cli 则把工具调用、终端操作和 ACP/MCP 放进同一个 Agent。今天真正值得看的，是权限、缓存、失败降级和结果回传。

🧱 标准化与低显存推理同时升温，工程界在补两种地基

apache/ossie 治理“数据含义”，airllm 治理“显存账本”。一个试图让 AI、BI 和分析工具共享语义模型，一个让大模型分层进出显存；都实用，也都需要在真实环境里验证兼容性和性能代价。

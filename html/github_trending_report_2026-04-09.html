An agentic skills framework & software development methodology that works.

📖 README 摘要

Superpowers 是一个完整的软件开发工作流框架，为你的编程代理（coding agents）构建，基于一组可组合的"技能"和初始指令。当代理启动时，它不会直接写代码，而是先退后一步问你真正想要做什么。在你签署设计方案后，代理会制定清晰的实施计划，然后启动子代理驱动的开发流程。Claude 通常可以自主工作数小时而不偏离计划。

✨ 核心特性

  • 自动触发的 agentic skills 框架
  • 子代理驱动的开发流程，支持数小时自主工作
  • 强调 TDD、YAGNI、DRY 开发原则
  • 支持 Claude Code、Cursor、Codex、OpenCode、Gemini CLI 等多平台
  • 可组合的技能系统，自动识别并应用

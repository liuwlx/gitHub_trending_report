📖 项目摘要

Catch2 是面向现代 C++ 的单元测试框架。它把断言、测试分段、BDD 风格描述和基础基准测试放在一套轻量接口里，适合希望降低测试样板代码、又不想引入庞大测试基础设施的 C++ 团队。

✨ 核心特性

  • 断言表达式直接、可读，失败时提供清楚的上下文
  • 通过 SECTION 复用测试准备逻辑，减少重复脚手架
  • 同时提供 BDD 宏与基础微基准，覆盖行为描述和性能回归
